# Empathetic Mood Assistant 🌿

Welcome to the Empathetic Mood Assistant! I built this project in a completely vibe-coded way on June 15, 2026—literally the first day of my free 5-day vibe coding course. Just going with the flow, trusting the process, and putting something meaningful together. 

This app is basically a chill, AI-powered companion. You just type out how you're feeling, and it uses Google's Gemini API to reflect back your mood, give you a little validation, and suggest a super quick micro self-care exercise. No judgment, just good vibes and a tiny moment of mindfulness when you need it.

## How to run this locally

If you want to spin this up on your own machine, it's pretty straightforward:

1. **Install the stuff:** 
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

2. **Add your API key:**
   Create a `.env` file in the root folder (you can use `.env.example` as a template) and drop your Gemini API key in there:
   ```env
   GEMINI_API_KEY=your_actual_key_here
   ```

3. **Fire it up:**
   ```bash
   npm run dev
   ```
   Then just open your browser to `http://localhost:3000` and you're good to go!
